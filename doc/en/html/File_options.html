Source files

This is where you can select files to be used as databases for file viewing (file lists with their descriptions), command descriptions and version file.

The default files usually reside in directory ~/.klininfo.

Table 3.1. Default KLinInfo files

Command Descriptions~/.klininfo/commandrc.kl
System Files~/.klininfo/systemrc.kl
Log Files~/.klininfo/logrc.kl
Program version file~/.klininfo/filever

KLinInfo uses special type of files with extension .kl. This is a kind of structured text files with entries, that consist of file name and description. The description is written in the next line of the word DESCRIPTION and may contain various HTML tags, which affect it's displaying.

This is an example of .kl file:

NAME: grep
DESCRIPTION: 
<p>   Searches for strings which have particular pattern, in files or in stdout.</p><p>   <i>Grep -e</i> is the same as 
<i>egrep</i></p><p>   <i>Grep -f</i> is the same as <i>fgrep</i></p>
<p>-c or --count options gives a possibility to print only a preferrable number of lines</p>
******************************************************************
NAME: vmstat
DESCRIPTION:
  Prints out the information and statistics of memory usage.
******************************************************************
NAME: amp
DESCRIPTION: 
<p>A console <b>mp3</b> player</p><p>It's usage is similar to <i>mpg123</i></p>
******************************************************************

The buttons in the right allow to edit those files directly with the editor that is set in Miscellaneous options in this dialog..

Note

Although it's possible to edit all these files, it's not recommended to edit description files (with extension .kl) directly. It's easier to use Add/Edit/Remove buttons in Command description window and file viewing windows.
