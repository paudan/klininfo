Font section

This is where you can get the list of TrueType fonts (TTF), and get a preview of them in almost every avilbale style and size. You can also save a list of their names as the examples in the HTML file. You can also define the string for preview in the Program Preferences dialog.

There are two tabs for fonts, one for KDE fonts, and one for the font families. Why? Well, it's because these two list may differ, but it should be the same.
