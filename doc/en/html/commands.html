Chapter 4. Menu Reference

The main KLinInfo window

This chapter briefly describes the KLinInfo menu.

The File Menu

File->Save->To separate files

Saves the information to separate HTML files in selected directory.

File->Save->To a single file

Saves the information to a single HTML file.

File->Quit (Alt-X)

Quits KLinInfo.

The Settings Menu

Settings->Edit files->Command descriptions file

Directly edit current command description file

Settings->Edit files->System files descriptions file

Directly edit current file with descriptions of system files

Settings->Edit files->Log files descriptions file

Directly edit current file with descriptions of log files

Settings->Edit files->Version file

Directly edit current file with commands for versions of particular programs (commands)

Settings->Edit files->Information script

Directly edit script all_info

Settings->Program preferences

Shows the Preferences dialog

The Help Menu

Help->Help (F1)

Shows this help window.

Help->What's this?

Lets you get short information by clicking on any object of the KLinInfo program window.

Help->About

Shows About KLinInfo window.

Help->About KDE

Shows About KDE window.
