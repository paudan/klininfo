Compilation and Installation

In order to compile and install KLinInfo on your system, type the following in the base directory of the distribution: 

% ./configure
% make
% make install

Since KApp uses autoconf and automake you should have not trouble compiling it. Automake v1.8 was used to create make scripts, but it should compile with earlier versions as well. Should you run into problems please report them to the author's mailing list.
