Files section

This window allows to save a list of special Linux files with texts, describing them. It's very similar to the command description dialog - you can add new files with their descriptions, edit them or remove (not the files, only the lines with their descriptions), except that you can also preview the contents of these files (if it is possible). The description can contain HTML tags (like <b> for bold text, <i> for italic, <p> for paragraph and etc.)

Log files window

The Log File preview window is meant for log files and their descriptions. The data is kept in ~/.klininfo/logrc.kl by default, and this can be changed in Preferences dialog.

System Files window

The System Files preview window shuould be used for various system configuration files and their descriptions. The data is kept in ~/.klininfo/systemrc.kl by default, and this can also be changed in Preferences dialog.
