The Linux Information Section

This section includes such types of information, as disk usage information, memory information, printenv information, process information, partition information, modules information and custom information, that could be read after running script, that could be easily edited by the user.

Disk Information window

This window is a graphical representation of du command. It allows you to display the information in different sizes (KB, MB, etc.)

Memory Information window

This is where you can find out about memory usage on your system. The results are represented in bytes, kilobytes and megabytes.

Other Information

This provides the output of a bash script. This script is named all_info and can be edited freely, either selecting Settings->Edit Files->Information Script, either manually. After install this file should reside in /usr/bin.

printenv window

This is where you can get all output of printenv.

Process Information window

This allows you to view information on processes, run on current machine. The process information can be displayed as user-oriented or normal styles, or either as signals or registers. Also there's a possibility to view either processes, run by current user (if Show only processes, run by current user is checked), either only running processes (if Show only running processes is checked), or combine both of them, by checking both checkboxes.

For more detailed about processesinformation about processes, refer to ps man page.

Partition Information window

Here you can view the information about your hard disks and partitions. This is the information, taken from fdisk and cfdisk, and can be displayed as:

With sizes in cylinders

Displays the partition tables with sizes of partitions as cylinders

With sizes in sectors

Displays the partition tables, giving their start and end sectors

In sector order format

Displays the partition tables in sector order format(the number of the partition, the partition type, the first sector, the last sector, the offset from the first sector of the partition to the start of the data, the length of the partition, the filesystem type (with the hex value in parenthesis), and the flags (with the hex value in parenthesis). Also free and unusable space is printed.

Partition table in raw format

Displays the partition tables in raw format( the number of the partition, the flags (in hex), the starting head, sector and cylinder, the filesystem ID (in hex), the ending head, sector and cylinder, the starting sector in the partition and the number of sectors in the partition

In raw format

Displays the partition data in raw format( the sectors that would be written to disk if a write command is executed. The primary partition table is printed at first, followed by the partition tables associated with each logical partition. The data is printed in hex with 16 bytes per line.

Module Information window

This gives the list of modules, currently used by Linux kernel.
