Miscellaneous options

Here are some other options that could also be set:

Show table grid

If this is checked, you will be able to see the grid oin the table widgets in applications.

Editor for custom file editing

Allows you to select your favorite editor for editing files, so youcan use kwrite, emacs and other editors.

Text caption, used in font dialog display

Define a string you want to see when you browse fonts in font display dialogs.
