The Packages section

Here you can get various information about Linux packages (RPM, packages, that are installed in Slackware), also get a program version list.

The RPM search window

This gives you the possibility to view available RPM packages. You can get information on RPM packages, that contain the your input string, and pressing List all packages will list all RPM packages, that are present on your Linux system.

Package Information window

This outputs the complete list of Slackware packages, installed on this Linux system, as well as their sizes, compressed and uncompressed. You can also preview the descriptions and the files that each package contains, in the tab window below.

Note

Note: this is only the list of Slackware packages (that is, only .tgz or .tar.gz packages. Packages, like Mandrake or Redhat RPM files, will not be listed here. You can get a list of them in the RPM packages window.

The RPM search window

This is where you can get versions of the programs defined in file ~/.klininfo/filever. Editing it is easy - - all that you need to define is program name and command which outputs the program version information, for e.g. for gcc you should write gcc --version .

An example of file filever: 

# This is a list of version commands. You can add your own to it.

gcc --version
php -v
python -V
xpdf -v
galeon --version
perl --version
