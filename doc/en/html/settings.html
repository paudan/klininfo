Chapter 3. KLinInfo settings

This chapter will describe the KLinInfo Program Preferences dialog. This dialog gives you an opportunity to customize the application in any way you want. You can change display, font and other options, select files tobe used as databases or change the default editor. It can be called by selecting Settings->Program Preferences in the Settings menu.

Font options

Just select the font, it's size and style for all the windows in your application. You can select different font for different widgets of KLinInfo.
