Chapter 2. Using KLinInfo

The Command Description window

This window allows to save a list of Linux commands and user-defined descriptions for them, so it's like a notepad for commands. You can add new Linux commands with their descriptions, edit them or remove. The description can be any size, and it can contain HTML tags (like <b> for bold text, <i> for italic, <p> for paragraph and etc.)

The information by default is stored in ~/.klininfo/commandrc.kl file, but if you wish to use another file, you can set it in Preferences dialog ( Settings->Program Preferences ), in the Files tab.
